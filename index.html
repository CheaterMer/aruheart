이용약관 (Terms of Service)

본 Discord 봇(이하 "서비스")은 사용자의 Discord 계정을 기반으로 다양한 기능을 제공합니다.

1. 서비스 이용

서비스는 무료로 제공되며, 사용자는 자발적으로 명령어를 실행함으로써 봇과 상호작용합니다.

2. 책임의 한계

서비스 운영자는 봇 사용 중 발생할 수 있는 문제나 피해에 대해 책임지지 않습니다.

3. 이용 제한

불법적인 용도로 봇을 사용하거나, 봇의 기능을 악용할 경우 사용 제한 또는 차단될 수 있습니다.

4. 약관 변경

운영자는 사전 통보 없이 약관을 변경할 수 있으며, 변경된 약관은 즉시 효력을 가집니다.

개인정보 처리방침 (Privacy Policy)

본 Discord 봇(이하 "서비스")은 사용자의 개인정보를 최소한으로 수집하며, 수집한 정보는 외부에 공유되지 않습니다.

1. 수집하는 정보

  • Discord 사용자 ID
  • Roblox 계정 정보 (연동된 경우)

2. 수집 목적

사용자 식별 및 봇의 기능 제공(명령어 응답, 역할 부여 등)을 위해 수집됩니다.

3. 정보 보관 기간

정보는 사용자가 연동을 취소하거나 봇에서 탈퇴할 때까지 보관됩니다.

4. 정보 공유

서비스는 수집된 정보를 제3자에게 제공하지 않으며, 마케팅 목적으로 사용하지 않습니다.
